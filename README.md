## Setup

Node.js must be installed

- `cd <project folder>`
- `npm install -g gulp`
- `npm install`

## Run tests

- `gulp init`
- `gulp develop`
- Open `localhost:3000/<test name>` (e.g. `http://localhost:3000/layout/`)
